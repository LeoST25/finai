# API

## Base URL

```
http://localhost:3000
```

---

## Endpoints

### GET /expenses

Lista despesas.

---

### POST /expenses

Cria uma despesa.

---

### PUT /expenses/:id

Atualiza uma despesa.

---

### DELETE /expenses/:id

Remove uma despesa.

---
