# 🚀 FinAI Roadmap

## Visão

O FinAI é uma plataforma de gestão financeira inteligente que integra um Dashboard Web moderno com um assistente financeiro via WhatsApp.

O objetivo do projeto é evoluir de um MVP para uma plataforma SaaS utilizando práticas modernas de Engenharia de Software.

---

# Status do Produto

## ✅ MVP

- [x] Backend NestJS
- [x] CRUD de despesas
- [x] CRUD de receitas
- [x] Prisma ORM
- [x] Dashboard React
- [x] React Query
- [x] Gráficos
- [x] Integração WhatsApp
- [x] Swagger
