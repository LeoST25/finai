# 📖 Sobre

O **FinAI** é uma plataforma de gestão financeira que permite registrar receitas e despesas de duas formas:

- 📱 Conversando pelo WhatsApp
- 💻 Utilizando um Dashboard Web moderno

O objetivo é evoluir para um **copiloto financeiro**, utilizando Inteligência Artificial para analisar gastos, gerar insights e auxiliar na tomada de decisões.

# ✨ Funcionalidades

## Backend

- API REST com NestJS
- Prisma ORM
- SQLite
- Swagger
- Integração WhatsApp (Baileys)
- CRUD de Receitas
- CRUD de Despesas

- Dashboard Financeiro
- Cards de KPI
- CRUD Manual
- Gráfico por Categoria
- Gráfico Mensal
- Últimas Transações
- Insights Financeiros
- Quick Actions
- React Query
